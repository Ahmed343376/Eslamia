# الإسلامية للموبيليا — الموقع الرسمي

موقع معرض الإسلامية للموبيليا، مبني بتقنيات HTML + CSS + JavaScript ونظام إدارة محتوى سحابي (Cloud CMS).

## 🚀 النشر على GitHub Pages

١. ارفع كل الملفات على GitHub Repository.
٢. اذهب إلى **Settings** ← **Pages**.
٣. اختر: **Source** ← **GitHub Actions**.
٤. اضغط **Save** — الموقع سينشر تلقائياً عند كل "Push".

## 🖼️ كيف تضيف صور المنتجات

١. ضع الصور في مجلد: `assets/images/products/`.
٢. أسماء الملفات: `bed-1.jpg`, `living-1.jpg`, إلخ.
٣. ارفع الصور على GitHub — وستتمكن من اختيارها وتغييرها من لوحة الإدارة.

## ⚙️ لوحة الإدارة السرية (Cloud CMS)

- افتح أي صفحة من الموقع.
- اضغط الاختصار: **`Ctrl + Shift + A`**.
- كلمة المرور: **`islamiya2024`**.
- **المميزات:**
    - تعديل النصوص والصور والروابط مباشرة (Visual Editor).
    - **Cloud Sync:** يتم حفظ التغييرات فوراً في سحابة جوجل (Firebase).
    - **نشر التغييرات:** اضغط زر "نشر التغييرات" لتظهر للجمهور فوراً.
